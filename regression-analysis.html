Regresión Lineal

Análisis estadístico · Pearson
Datos de entrada
x y
Media de x
x̄ = (Σxᵢ) / n
—
Media de y
ȳ = (Σyᵢ) / n
—
Covarianza
σxy = Σ(xᵢ-x̄)(yᵢ-ȳ) / n
—
Correlación Pearson
r = σxy / (σx · σy)
—
Ecuación de la recta de regresión
m = σxy / σx² · n = ȳ − m·x̄
ŷ = m·x + n
Diagrama de dispersión
n = 0
